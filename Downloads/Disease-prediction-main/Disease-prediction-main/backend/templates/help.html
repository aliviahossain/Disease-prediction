Glossary / Help Section

Understand the key terms used in probability and diagnostic calculations.

What is Prior Probability?+

It’s the probability of a condition or event before new evidence (like a test result) is considered. For example, the chance someone has a disease before any medical test. Learn more →

What is Sensitivity?+

Sensitivity measures how well a test identifies true positives. A test with 95% sensitivity correctly detects 95 out of 100 people who have the disease. Learn more →

What is Specificity?+

Specificity measures how well a test identifies true negatives. A 90% specific test correctly clears 90 out of 100 healthy people. Learn more →

What is False Positive Rate?+

It’s the percentage of people incorrectly marked as positive when they are actually negative. Calculated as (1 - Specificity). Learn more →

What is Posterior Probability?+

The updated probability of a condition after considering new evidence, like test results. It’s calculated using Bayes’ theorem. Learn more →
